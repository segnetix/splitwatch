Splitwatch

Splitwatch is a utility designed to give you an automatic calculation of rolled up splits as you are timing. That is, for say 200m intervals, Splitwatch will give you each 200, 400, 800, and 1600 (or 1000m if in Kilo mode) split for quick comparison/analysis of various parts of a race. Splitwatch is designed especially for Track and Swimming training and competition but is also effective for any event where split interval recording is needed.

Lap Distance Indicator

Indicates the current lap distance setting. Each time you press the lap button the Splitwatch will record a lap of that distance. Distances can be either metric or English units with intervals of 25, 50, 100, 200, or 400 meters or 25, 50, 110, 220, 440 yards (see Settings: Split Units for more info).

Lap Count

Indicates the number of completed laps.

Current Lap Time

Keeps track of the time for the current lap. The lap time is frozen for 5 seconds each time the lap button is hit. After that it resumes tracking the current lap time.

Main Clock

Keeps the running time from start to stop, up to 99:59:59.99 (that is 4 days 3 hours 59 minutes 59.99 seconds).

Start/Stop Button

Starts the Splitwatch and changes to a Stop button. Time will continue to be tracked if you move to other parts of the interface, quit the application or even power down your iPhone/iPod Touch, until the Stop button is hit.

Lap/Reset Button

Records a lap event each time the Lap button is hit while the Splitwatch is running. Note that if the Lap Disable control is on (see Settings: Lap Disable Control), the Lap button will be disabled for 2 seconds to prevent accidentally hitting the Lap button more than once per lap. Turn the Lap Disable control off to have the Lap button enabled immediately. The Lap button changes to the Reset button after the Splitwatch is stopped.

Split Column Header

Indicates the current split columns. This column header will change with the split unit setting (see Settings: Split Units).

Split Detail View

Each interval will be represented as a row with appropriate splits rolled up to the right. While running the split detail view will automatically scroll as needed. After the Splitwatch is stopped the split detail view can be scrolled manually to review all the splits.
