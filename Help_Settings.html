Settings

The Settings view holds your application preferences.

Split Unit/Interval Selector

The Split Unit/Interval Selector sets the base unit for Splitwatch laps. The unit can be Metric, English, or Lap. The intervals for each lap can be 25m, 50m, 100m, 200m, or 400m for Metric units, or or 25y, 50y, 110y, 220y, or 440y for English units. The interval distance does not apply for Lap units, so the interval selector is disabled with that selection. Changes to the split units will be used for the next Splitwatch or Multiwatch event.

Kilo Splits Switch

The Kilo Splits switch will enable the kilo split mode. In this mode each 1000m interval will be calculated. Note that Kilo Split mode is available only with 200m split intervals.

Furlong Mode

Furlong mode is for horse racing. Distances are marked in fractions of a mile instead of yards or meters. Note that Furlong mode is available only with English 220y or 440y split intervals.

Sound Switch

The Sound switch enables or disables a click sound for each tap of the Start/Stop/Lap/Reset buttons.

Lap Disable Switch

The Lap Disable switch activates a 2 second period after hitting a Lap button, during which the Lap button is disabled. This is to reduce the possibility of accidentally hitting the Lap button more than once while recording a lap.

Start/Stop and Lap/Reset on Lift Switches

Enabling these switches changes the behavior of the Start/Stop and Lap/Reset buttons so that the action is activated when your finger lifts from the button instead of when it touches the button. This allows you to touch and hold the button while waiting for the start/stop/lap, then lift without having to look at the watch to capture the time. If you set the Lap/Reset on Lift switch it might be helpful to turn off the Lap Disable switch. Set the Lift switches to Off for the start/stop/lap/reset buttons to function in the standard manner.

Email Setup

The Email button brings up the Email Setup view. Here you can assign default email addresses that will be placed in the To: field each time you bring up the Split Report email interface.

Add Contact Button

Brings up the Contacts interface to select recipients from your contacts list. Select a contact to add it to the contacts list.

Contact Text Field

Tap the Contact Text Field to display the keyboard. Here you can enter email recipients that are not in your contacts list.

Contacts List

This is the current list of contacts that will be placed in the To: field of your Split Report email interface.

Edit/Done Buttons

Puts the current contact list into edit mode. In edit mode you can rearrange the order or delete items from the email contact list. Hit the Done button to exit edit mode.

Settings

Returns you to the Settings view.

About

General information about Splitwatch, including the current version, credits, contacts, and your event database stats.

Clear All Timing Data

This button will ask if you want to delete all Splitwatch timing data.
