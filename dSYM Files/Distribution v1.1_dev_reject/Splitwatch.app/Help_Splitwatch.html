Splitwatch

Splitwatch is a timing utility designed to give you an automatic split breakdown as you are timing. For example, if timing 200m splits, Splitwatch will give you each 200, 400, 800, and 1600 (or 1000m if in Kilo mode) interval for quick comparison and analysis of various parts of a race. Splitwatch is designed especially for Track and Swimming training and competition but is also effective for any event where interval recording is needed.

Lap Distance Indicator

Indicates the current lap distance setting. Each time you press the lap button Splitwatch will record a lap of that distance. Lap distances can be either metric or English units with intervals of 25, 50, 100, 200, or 400 meters or 25, 50, 110, 220, or 440 yards (see Settings: Split Units Selector for more info).

Lap Count

Indicates the number of completed laps.

Lap Time

Keeps track of elapsed time for the current lap. The lap time is frozen for 5 seconds each time the lap button is hit. After that it resumes tracking the current lap time.

Main Clock

Keeps the running time from start to stop, up to 99:59:59.99.

Start/Stop Button

Starts the Splitwatch and changes to a Stop button. Time will continue to be tracked in the background if you move to other parts of the application, quit Splitwatch or even power down your iPhone/iPod Touch, until the Stop button is hit.

Lap/Reset Button

Records a lap event each time the Lap button is hit while the Splitwatch is running. Note that if the Lap Disable control is on (see Settings: Lap Disable Control), the Lap button will be disabled for 2 seconds to prevent accidentally hitting the Lap button more than once per interval. Turn the Lap Disable control off to have the Lap button enabled immediately. The Lap button changes to the Reset button after the Splitwatch is stopped.

Split Column Header

Indicates the current split columns. This column header will change with the split unit setting (see Settings: Split Units Selector).

Split Detail View

Each split will be represented as a row with all intervals calculted to the right. The split detail view will automatically scroll as laps are added. After the Splitwatch is stopped the split detail view can be scrolled manually to review all the timing data.

Last Splits

At the bottom of the Split Detail View is the Last split row. The Last splits display closing times over the last 5 splits.
