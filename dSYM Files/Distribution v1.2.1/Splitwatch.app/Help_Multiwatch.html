Multiwatch

Multiwatch is designed to provide you with up to 5 simultaneous Splitwatches. All can be started together or each watch can be started independently.

Lap Distance Indicator

Indicates the current lap distance setting. Each time you press the lap button the Splitwatch will record a lap of that distance. Distances can be either metric or English units with intervals of 25, 50, 100, 200, or 400 meters or 25, 50, 110, 220, 440 yards (see Settings: Split Units Selector for more info).

Start Button/Main Clock

Starts all the Multiwatches and changes to the running time. Time will continue to be tracked if you move to other parts of the interface, quit the application or even power down your iPhone/iPod Touch, until all the Stop buttons are hit.

Reset

After all the watches are stopped, tap the Main Clock to reset.

Multiwatch Cell

Each Multiwatch has its own cell.

Start/Stop Button

Starts a single Multiwatch and changes to a Stop button. Also starts the main clock if not already running.

Lap/Reset Button

Records a lap event each time the Lap button is hit while the Splitwatch is running. The Lap Disable control (see Settings: Lap Disable) activates the disable delay function of the Lap buttons if enabled. The Lap button changes to the Reset button after the Splitwatch is stopped.

Athlete Name

Displays the name associated with a Multiwatch. The name is set via the Info button (see Multiwatch: Info Button below).

Lap Count

Indicates the number of completed laps for a Multiwatch.

Lap Time

Keeps track of elapsed time for the current lap for a Multiwatch. The lap time is frozen for 5 seconds each time the lap button is hit. After that it resumes tracking the current lap time.

Total Time

Keeps the running time for a Multiwatch from start to stop, up to 99:59:59.99.

Setting Up the Multiwatches with the Info Button

The Info button flips the Multiwatch view over to the Multiwatch setup view (note: the Info button is disabled while the watches are running). Athlete name and event name is recorded with the split data when the Stop button is hit. This information will be used by the History view to identify an event (see History: Sort Tab Bar Control and History: Split Detail).

Athlete Name Fields

Tap an Athlete Name field to bring up a keyboard or use the Name Selector Button to set the name and activate a Multiwatch. Up to five Multiwatches can be activated.

Event Name Field

Tap the Event Name text field to assign an event name for each of the activated Multiwatches.

Name Selector Buttons

These buttons bring up a selection wheel with all the athlete or event names in the database. Select a name and tap the Set button to quickly assign a name to a Multiwatch.

Done Button

Tap the Done button to return back to the Multiwatch view.
