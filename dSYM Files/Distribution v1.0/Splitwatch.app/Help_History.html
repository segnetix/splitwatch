History

The History view is used to recall all of your recorded Splitwatch events and split data.

Filter Tab Bar

The Filter Tab Bar helps you easily recall a specific Splitwatch event.

All

The All tab control lists every recorded Splitwatch event by date with the most recent first. Select an event to display the event and split data (see History: Split Detail View below).

Date

The Date tab control will list all of the dates on which you recorded Splitwatch events. Most recent appear first. Select a date to display a list of events from that date.

Distance

The Distance tab control lists all of the recorded event distances from shortest to longest. Select a distance to display a list of events of that distance.

Event

The Event tab control lists all of the recorded event names alphabetically. Select an event name to display a list of events with that name.

Athlete

The Athlete tab lists all of the recorded athlete names alphabetically. Select an athlete name to display a list of events by that athlete.

Edit Button

The edit button is used to delete events or groups of events from the History database. To delete an event or group of events, tap the Edit button to put the current list into edit mode. Optionally, left swipe a row to bring up the delete button directly. Tap the Done button to exit edit mode.

Split Detail View

Selecting an event in the History view brings up the Split Detail view. The Split Detail view displays all of the data recorded for that event.

Athlete/Event Name Editing

Select the Athlete and Event text fields to assign or edit the names for this event. If this event was recorded by a Multiwatch then the text fields will display the names assigned to the Multiwatch.

Event Info

Date: The date and time the event started

Distance: Total distance of the event

Time: Total time for the event

Split Column Header

Indicates the split columns for the event. This column header will change depending on the split unit selected for the event (see Settings: Split Units Selector).

Split Detail

Each interval will be represented as a row with appropriate splits rolled up to the right. The split detail view can be scrolled if necessary to view additional rows.

History Button

The History button returns to the History view.

Email Composer

The Email composer allows you to easily export event data by emailing a Split Report.

To/CC/Bcc: Assign the recipients by tapping the To/Cc/Bcc fields. This will display a keyboard and the Add Contact button. The Add Contact button lets you select an email address from your list of contacts. If you have added contacts in the Email Setup view then those addresses will be placed in the To field when the Email Composer view is displayed.

Subject: The subject contains the name of the athlete, the event distance and the final time. The subject line can be edited before sending.

Split Report: The body of the email is the Split Report with all the event info and split data. The body of the Split Report can be edited before sending.

Send/Cancel: Hit the Send button to send the email to the recipients or Cancel to cancel and dismiss the Split Report.
